# QuikVite

**QuikVite** is a simple CLI to quickly create and run a Vite React+TypeScript project.

## Features

- Creates a new Vite React+TypeScript project
- Automatically installs dependencies
- Automatically starts server

## Installation (Global)

```bash
npm install -g quikvite
```

## Usage (Global)

Replace `my-app` with desired project name.

```bash
quikvite my-app
```

## Local Usage:

Replace `my-app` with desired project name.

```bash
cd quikvite
node quikvite my-app
```
